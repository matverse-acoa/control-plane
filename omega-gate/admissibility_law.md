# Lei de Admissibilidade Científica

Uma submissão só é **PASS** quando:

- hipótese é formal;
- protocolo é auditável;
- protocolo está completo;
- rastreabilidade está presente;
- hash de evidência foi informado.

A ausência de rastreabilidade ou hash pode levar a `ESCALATE` para revisão constitucional,
conforme parametrização de regras.
